## Description ##
Upload new vacation destination and track their popularity

## Usage ##
1. Cloaning the repo into your code editor.
2. 'Copy-Paste' the 'build_data' file into MySQL and run it.
3. 'Copy-Paste' the 'insert_data' file into MySQL and run it.
4. Open local port 3306 for the connection between the server and the database.
5. Go to the server folder and run the index.js file from the server (can be done by running node index.js in the terminal).
6. Go to the client folder and run command npm start.
